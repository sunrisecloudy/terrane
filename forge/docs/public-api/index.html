Overview

Forge exposes one command facade (WorkspaceCore::handle) to every shell, agent, CLI, and HTTP bridge. Generated applets never call that facade directly — they use the typed ctx.* host API declared in forge/std/forge-std.d.ts.

Applet authors

Import @forge/std, export main(ctx, input), declare capabilities in manifest.json.

Operators

Use forge run, the web console at /console, or POST /bridge.

Integrators

Pin artifacts/public-contract.json and verify with tools/verify-public-contract.mjs.

CLI

Binary: forge from forge-cli. Every outer command is reachable through forge run <name>.

forge commands

List catalog commands (tier/namespace filters, --json).

forge describe <name>

Show one command descriptor and schema paths.

forge run <name>

Execute any outer command locally or via --server.

forge trace <run_id>

Read redacted host-call journal via system.trace.

forge demo

M0a spine: install notes-lite, run, replay byte-identically.

forge help

Catalog-generated help grouped by namespace.
